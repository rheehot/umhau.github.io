This is my source, modified for OpenBSD.  Using it because my computer is basically running headless.  http://www.dnacoil.com/tools/setting-up-syncthing-for-raspberry-pi/

This involves two computers.  I'm using my trusty Mint 18 installation on one end, and my fiesty OpenBSD 5.9 (yes, it's outdated) installation on the other.

On Mint 18, install syncthing and get the browser-based GUI running:
curl -s https://syncthing.net/release-key.txt | sudo apt-key add -
echo "deb http://apt.syncthing.net/ syncthing release" | sudo tee /etc/apt/sources.list.d/syncthing.list
sudo apt-get update
sudo apt-get install syncthing
syncthing &
If it worked, your browser will open automatically and show you the control panel for your shared folders.  You're done for now.  Time to ssh into your OpenBSD installation and get Syncthing running there.
