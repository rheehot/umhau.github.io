Note:

I'm afraid the only "easy" way to change distros on early model Chromebooks relies on borrowing the original ChromeOS kernel as described here. If you want a different kernel you will have to build and sign it yourself.

If you're willing to install ordinary x86 Arch you can put it together in a VM, partition the SSD and then copy the Arch partition over raw. The page linked above goes into detail about how to do this and sub in the ChromeOS kernel.
